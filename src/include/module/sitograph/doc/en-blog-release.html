Sitograph Development Team is proud to introduce the stable release of the Sitograph Content Management System 1.0.

Sitograph is open source Content Management System that enables you to build websites and efficient online applications.

Modules and extensions for Sitograph CMS can be downloaded directly from Admin UI (Section 'Module Settings'). To browse repository catalog and write feedback go to rep.msvhost.com

To contribute in development please send module update as *.zip archive directly to Sitograph Repository using your personal key or commit to main GitHub repository directly ('development' branch).

To receive personal key please contact Sitograph Repository Support

The architecture of the service, a design of the application, core development and release was done by github.com/maxsv0 (Max Svistunov).

Any found bugs and suggestions please send to Sitograph Support


External libraries that were used in this release:

  • Bootstrap 3
  • jQuery
  • Smarty
  • Google APIs client
  • Fancybox
  • TinyMCE
  • Cropper JS
