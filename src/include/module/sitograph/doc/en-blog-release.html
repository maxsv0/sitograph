Sitograph Development Team is proud to introduce the stable release of
the Sitograph Content Management System 1.0.

Sitograph is open source Content Management System that enables you to build websites and efficient online applications.

Modules and extensions for Sitograph CMS can be downloaded from our repository rep.msvhost.com.

To contribute in development please send module update as *.zip archive directly to Sitograph Repository using your personal key or commit to main Git repository directly.

To receive personal key please contact Sitograph Repository Support

To contribute in development please send your module as *.zip archive to Sitograph Repository or commit to main Git repository directly.

The architecture of the service, a design of the application, core development and release was done by Max Svistunov.

Any found bugs and suggestions please send to Sitograph Support


External libraries that were used in this release:

  • Bootstrap 3
  • jQuery
  • Smarty
  • Google APIs client
  • Fancybox
  • TinyMCE
  • Cropper JS
