Sitograph Development Team is proud to introduce the stable release of the Sitograph Content Management System 1.0.

Sitograph is open source Content Management System that enables you to build websites and efficient online applications.

Modules and extensions for Sitograph CMS can be downloaded directly from Admin UI, {HOME_URL}admin/.

To browse list of available modules visit the Sitograph Repository

For any found bugs or suggestions please raise a issue at GitHub, https://github.com/maxsv0/sitograph/issues

To contribute in development please contact Sitograph Repository Support

The architecture of the service, a design of the application, core development and release was done by Max Svistunov, github.com/maxsv0.


External libraries that were used in this release:

  • Bootstrap 3
  • jQuery
  • Smarty
  • Google APIs client
  • Fancybox
  • TinyMCE
  • Cropper JS
