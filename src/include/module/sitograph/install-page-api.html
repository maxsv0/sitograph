Sitograph API

Each module can implement an extension for API calls.

API output is encoded with JSON.

Example output:

{
	"ok":true,
	"data":[],
	"msg":"Empty result"
}

API functions installed by default

URLModulePHP FunctionDescription
/api/blog/list/ blog ajaxBlogRequest() List of published articles
/api/blog/category/ blog ajaxBlogRequest() List of categories for articles
/api/blog/details/12345/ blog ajaxBlogRequest() Details about article ID 12345
/api/gallery/list/ gallery ajaxGalleryRequest() List of published albums
/api/gallery/album/12345/ gallery ajaxGalleryRequest() List of photos for an album ID 12345
/api/feedback/list/ feedback ajaxFeedbackRequest() List of published feedback items
/api/feedback/add/ feedback ajaxFeedbackRequest() Publish new feedback.
* Note. User level access required
/api/uploadpic/ msv-core ajaxUploadPicture() Upload file to file storage.
* Note. Admin level access required
/api/more_search/ search ajax_More_Search() Get website text search results
