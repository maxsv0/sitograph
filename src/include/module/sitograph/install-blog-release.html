Sitograph Team is proud to introduce the first stable release of Sitograph CMS 1.0.beta

Sitograph is open source Content Management System that comes with an integrated development kit that allows extending the functionality of your website.

Catalog of free Sitograph modules can be found at rep.msvhost.com repository

To contribute in development please send your module as *.zip archive to Sitograph Repository or commit to main Git repository directly.

The architecture of the service, a design of the application, core development and release was done by Max Svistunov.

Any bugs and suggestions please send to Sitograph Support


External libraries that were used in this release

  • Bootstrap 3
  • jQuery
  • Smarty
  • Google APIs client
  • Fancybox
  • TinyMCE
  • Cropper JS
