Application Kernel contains: msv-core, msv-api, msv-users, msv-seo.

Kernel modules provides website with basic functionality.

msv-core is a main website module. It provides with a set of functions to handle website requests, store website structure, allocate resources, install and update of new modules.

msv-api is a data connector module. It provides with data exchange functions that allows to run SQL queries, update rows, delete and create database tables.

msv-users is a module to handle website users. Allows common functionality: register new user, log in, reset password, update personal information, email notifications.

msv-seo provides Search Engine Optimization for website. Allows to edit common page META headers and track information about visitors.
